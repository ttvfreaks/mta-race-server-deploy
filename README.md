# mta-race-server-deploy

Изначально разрабатывался для ивента стримера boryacreative - TAA

## Запуск:

Делаешь сервер на Ubuntu, заходишь туда по ssh и выполняешь команду:
```
bash <(curl -sL https://raw.githubusercontent.com/ttvfreaks/mta-race-server-deploy/refs/heads/main/deploy.sh)
```

## Процесс установки:

Надо будет ввести пароль, по которому нужно будет заходить на сервер

После быстрой установки сервера (~30 сек) в консоли появится ссылка на подключение к игре, которую можно просто открыть в браузере
